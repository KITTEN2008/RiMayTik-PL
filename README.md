# RiMayTik-PL
RiMayTik PL(сокращенно Ri) - это русскоязычный язык программирования, ориентированный в первую очередь на новичков.

Для данного языка создана собственная среда разработки(IDE) в файле ri_ide.py - запустив его Вы можете начать писать программы сразу же или ознакомившись с командами и примерыми. Для этого в IDE есть вкладка "помощь", где описаны команды для различных функций, а также последние обновления языка и IDE.
